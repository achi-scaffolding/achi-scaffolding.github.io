Components & Reusable Parts
Header & Footer
Stored in /partials/
Automatically loaded in every page using:
loadHeader.js
loadFooter.js
Editing header/footer updates all pages.

Sliders
Hero slider (homepage): slider.js
Clients slider: clients-slider.js
Testimonials slider: testimonials-slider.js
Why Choose Us slider: why-slider.js

 Blog System
Blog cards on the homepage load dynamic dates using blog-dates.js
Clicking a card opens a dedicated blog-post HTML page.
Blog layout uses simple reusable card style.

Responsiveness
All pages are being designed to support:
Desktop
Tablet
Mobile (in progress)

 How to Work on the Project
Clone the repository
Open with any code editor ( VS code )
Edit HTML files normally
Modify header/footer once (partials folder)
Commit & push changes to GitHub
GitHub Pages will auto-update the live version
